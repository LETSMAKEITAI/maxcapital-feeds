# MAXCAPITAL — XML-фиды объявлений

Публичные фиды коммерческой аренды MAXCAPITAL для автозагрузки на площадки.
Здесь только XML-объявления (публичные данные). Кода бота тут нет.

| Площадка | Формат | URL фида |
|---|---|---|
| ЦИАН | ЦИАН feed v2 | `https://raw.githubusercontent.com/LETSMAKEITAI/maxcapital-feeds/main/cian.xml` |
| Яндекс.Недвижимость | YRL | `https://raw.githubusercontent.com/LETSMAKEITAI/maxcapital-feeds/main/yandex.xml` |
| Авито | Автозагрузка (Ads) | `https://raw.githubusercontent.com/LETSMAKEITAI/maxcapital-feeds/main/avito.xml` |

Источник — лист «📢 Объявления — контроль» файла Бориса; собирает бот REALESTATEBOT.
Обновление — раз в сутки. Публикация на площадках требует фото объектов.
